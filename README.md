# Sidekick

An Obsidian plugin.

## Development

- Install dependencies: `npm install`
- Dev build (watch mode): `npm run dev`
- Production build: `npm run build`
- Lint: `npm run lint`

## Manual installation

Copy `main.js`, `styles.css`, and `manifest.json` to your vault at `VaultFolder/.obsidian/plugins/sidekick/`.
